# اسکریپت چت روم فارسی رایگان والْت چت

نصب و راه اندازی:
- فقط کافیه فایل ها رو در هاستتون کپی کنید و تمام.

نکته:
یک فونت فارسی دلخواه با فرمت woff رو هم باید در پوشه اصلی کنار بقیه فایل ها قرار بدید،
و نام فایل رو به persianfont.woff تغییر بدید. میتونه هر فونتی باشه، وزیرمتن یا شبنم پیشنهاد میشه.

بکآپ گرفتن:
- فایل ها و پوشه ها رو کپی یا دانلود کنید، دیتابیس sqlite هست و تمام اطلاعات کپی میشه.

امنیت و رمزنگاری:
- این اسکریپت یک لایه امنیتی داره و تا زمانی که کلید(رمز) اتاق رو در اینترنت منتشر نکنید یا از راه امن به دیگران بدید اطلاعات شما رو کسی نمیبینه حتی سرور. اما دقت داشته باشید که این اسکریپت با هوش مصنوعی ساخته شده و احتمال داشتن باگ در اون زیاده. پس برای گفتگوهای خیلی حساس ازش استفاده نکنید.
